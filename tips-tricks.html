Tip #1: Optimizing Images for the Web

In order to optimized my original profile image, I first used a local software called PhotoScape X to resize my original image from 626x626px to 219x219px, then I optimized my image by using the online service https://tinyjpg.com/ which saved me 6.2KiB (a 3% reduction)
